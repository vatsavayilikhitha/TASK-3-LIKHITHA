# AI Movie Recommendation System

## Overview
This project is an AI-powered movie recommendation system that recommends movies using cosine similarity and TF-IDF vectorization.

## Technologies Used
- Python
- Pandas
- Streamlit
- Scikit-learn

## Features
- AI recommendation engine
- Cosine similarity matching
- Interactive UI
- Movie descriptions
- Similarity scoring

## How to Run

streamlit run app.py
